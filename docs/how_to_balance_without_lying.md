# Cómo balancear sin mentir clínicamente

No modificar la cobertura real para ajustar dificultad.

Modificar solamente variables de juego:

- costo;
- potencia;
- cadencia;
- rango;
- velocidad del proyectil;
- disponibilidad;
- tiempo de recarga;
- recursos del nivel;
- carga de toxicidad contextual.

La verdad clínica y el balance deben permanecer en estructuras separadas.
