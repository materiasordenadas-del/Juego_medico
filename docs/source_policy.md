# Política de fuentes

Las reglas clínicas deben apoyarse prioritariamente en:

1. guías de sociedades científicas;
2. documentos regulatorios y organismos sanitarios;
3. revisiones sistemáticas y literatura primaria;
4. textos farmacológicos y microbiológicos reconocidos;
5. antibiogramas y epidemiología local cuando el escenario lo requiera.

Cada regla debe enlazarse a una entrada de `src/data/evidenceSources.js`.
No se usarán blogs, Wikipedia ni contenido dirigido a pacientes como fundamento clínico.
