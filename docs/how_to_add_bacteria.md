# Cómo agregar una bacteria

1. Añadir el microorganismo en `src/data/bacteria.js`.
2. Definir cobertura relevante y resistencia basal.
3. Añadir caveats microbiológicos.
4. Registrar fuentes.
5. Crear pruebas de las interacciones principales.
6. Añadir el asset visual sin modificar el motor clínico.
